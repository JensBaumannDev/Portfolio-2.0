Personal portfolio website of Jens Baumann – built with Angular.

## Concept & Implementation

The design, concept and visual direction of this site are entirely my own.
AI tools were used purely as a development aid to implement those ideas more
efficiently.

## Tech Stack

- **Angular 21** (standalone components)
- **TypeScript**
- **SCSS**
- **@ngx-translate** – internationalization (DE/EN)
- **@lucide/angular** / **@ng-icons** – icons
- **Vitest** – testing

## Development

Requires Node.js and npm.

```bash
npm install        # install dependencies
npm start          # start dev server (opens http://localhost:4200)
npm run build      # create a production build
npm test           # run tests
```

## Features

- Responsive design
- Language switch DE/EN with persisted preference
- CV as PDF download (DE & EN)
- Project overview with links
